## Intro

We are generally exposed to `reduce` with an example to “reduce a collection of numbers to their sum.” So they say we “reduce something larger (a collection) to something smaller (a single resulting number).”

That is a good first example, but the concept is not really that we “reduce” in the sense of make smaller. We do, but not always, and that is not the main or sole purpose of “reduce”.

We can have a collection, and end up with a collection of the same size but modified in some way (like `map`). Or we can filter in our out results, like if we were using `filter`.

But it may also happen that we end up with an even larger collection than the one we started.
