## Intro

The identity property or the neutral value of an operation is a value that when used in the operation, does not change the result of the operation or some computation.

## Addition

When 0 is added from an expression, the value of the expression doesn't change. Observe:

- $7 + 0 = 7$
- $0 + 7 = 7$

Which we can generalize to:

- $x + 0 = x$
- $0 + x = x$

Because addition is commutative, 0 can be anywhere in the expression.
