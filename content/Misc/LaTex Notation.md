## Intro

Some notation for symbols I need to use from time to time.


## Equality symbols

| Read as          | Symbol               | Notation             |
| ---------------- | -------------------- | -------------------- |
| equal            | $=$                  | `=`                  |
| not equal        | $\neq$               | `\neq`               |
| should equal     | $\stackrel{?}{=}$    | `\stackrel{?}{=}`    |
| should not equal | $\stackrel{!}{\neq}$ | `\stackrel{!}{\neq}` |

- Equal: $=$ , `=`.
- Not equal: $\neq$, `\neq`.
- Should equal: $\stackrel{?}{=}$ , `\stackrel{?}{=}`
