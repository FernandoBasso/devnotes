## Search for text in the current file

Select some word or text, and then hit `Ctrl+f` or `Ctrl+f` to perform a search of that selection in the current file.

![VS Code search selection in current file](../__assets/vscode-search-selection-in-current-file.png)

## Search for text in whole project

Select some word or text, and then hit `Ctrl+Shift+f` or `Cmd+Shift+f` to perform a search of that selection in the whole current project. Then use `F4` or `Shift+F4` to navigate to the next and previous matches.

![VS Code global text search](../__assets/vscode-global-text-search.png)
